# Thought Virus：多智能体系统中的"思想病毒"攻击——单个智能体的潜意识偏见如何病毒式传播

📌 **核心摘要**

本文提出了一种名为"Thought Virus"（思想病毒）的新型攻击向量，揭示了多智能体系统（Multi-Agent Systems, MAS）中一个此前未被探索的安全漏洞。研究发现，通过潜意识提示（Subliminal Prompting），单个被攻陷的智能体可以将偏见传播至整个网络，即使后续智能体从未直接接收恶意输入。实验在 6 智能体链式拓扑中进行，结果显示：潜意识偏见可传播至网络末端，在 TruthfulQA 基准测试中，单个智能体的潜意识攻陷可导致下游智能体真实性下降。该攻击无需访问模型权重，仅通过系统提示词即可实现，且能逃避现有的 paraphrasing 和检测类防御机制。代码已开源：https://github.com/Multi-Agent-Security-Initiative/thought_virus

---

## 一、研究背景与动机

### 1.1 多智能体系统的安全挑战

大型语言模型（LLM）构成的多智能体系统（MAS）已在自主交易、协作编程等复杂任务中展现出超越单模型的性能。然而，这种架构在实现专业化和可扩展性的同时，也引入了新的攻击面。特别是**错误传播**（Error Propagation）现象——单个智能体的错误或偏见行为可能通过智能体间交互传播至整个系统，放大其影响。

### 1.2 现有攻击与防御的局限性

当前针对多智能体系统的攻击主要分为两类：

| 攻击类型 | 原理 | 防御方式 | 局限性 |
|---------|------|---------|--------|
| 对抗性提示（Adversarial Prompts） | 优化无意义后缀以触发特定行为 | 改写（Paraphrasing） | 依赖精确措辞，无法在智能体间传播 |
| 提示注入（Prompt Injection） | 通过角色扮演或欺骗绕过安全机制 | 消息监控与过滤 | 语义明确，可被自动检测 |

**核心问题**：是否存在一种攻击，既能逃避改写防御（不依赖精确措辞），又能逃避检测防御（不包含显式恶意内容）？

### 1.3 潜意识学习的启示

近期研究（Cloud et al., 2025; Zur et al., 2025）发现，语言模型可通过包含语义无关 token 序列的提示被偏见化，产生对具体概念（如偏爱某种动物）的系统性偏好，尽管提示与目标概念之间无明显语义联系。然而，这一现象此前仅在孤立的用户 -LLM 交互中被研究，其在多智能体系统中的影响尚属空白。

## 二、核心方法

### 2.1 Thought Virus 攻击框架

本研究提出**Thought Virus**，一种利用潜意识提示在多智能体系统中传播偏见的新型攻击向量。其核心思想是：若单个智能体被潜意识提示偏见化，该偏见将传播至与其通信的其他智能体，并由这些智能体进一步传播。

![Thought Virus 攻击示意图](https://arxiv.org/html/2603.00131v1/x1.png)

**攻击流程**：
1. **攻陷 Agent0**：通过系统提示词将潜意识 token（如三位数字"613"）与目标概念（如"lion"）隐式关联
2. **传播偏见**：Agent0 与 Agent1 对话，Agent1 与 Agent2 对话，依此类推
3. **病毒式扩散**：即使 Agent0 不再参与后续对话，偏见仍通过中间智能体自主传播

### 2.2 实验设置

**网络拓扑**：
- **单向链**（Chain）：A→B→C→D→E→F
- **双向链**（Bidirectional Chain）：A→B→C→D→E→F→E→D→C→B→A

**模型配置**：
- Qwen2.5-7B-Instruct（主实验）
- Llama-3.1-8B-Instruct（验证实验）
- 6 智能体网络，20 次随机种子重复

**潜意识 Token 选择**：
对 1000 个可能的三位数 token（000-999）进行单独评估，选择在目标行为上产生最强效应的 token 作为潜意识 token。

### 2.3 评估场景

#### 场景 1：动物偏好（Animal Preferences）
- **目标**：偏见化智能体输出特定目标动物
- **指标**：响应频率（200 次模拟响应中包含目标动物的比例）、目标动物的 log 概率
- **验证**：确保目标动物（如"lion"）未在任何智能体间消息中显式出现

#### 场景 2：真实性对齐（Misalignment / TruthfulQA）
- **数据集**：TruthfulQA-MC（684 道选择题，38 个类别）
- **目标**：评估潜意识 token 是否影响智能体回答真实性问题的倾向
- **指标**：准确率、正确与错误响应 token 的 log 概率差

## 三、实验结果

### 3.1 动物偏好传播效果

**关键发现**：潜意识偏见可在整个网络中传播，效应随距离衰减但仍显著。

| 智能体 | Lion（最强 token） | Lion（平均 token） | 随机 token 基线 |
|--------|-------------------|-------------------|----------------|
| Agent 0 | 1,386.4× | 427.0× | 1.0× |
| Agent 1 | 24.7× | 6.3× | 1.0× |
| Agent 2 | 23.9× | 5.0× | 1.0× |
| Agent 3 | 7.5× | 3.3× | 1.0× |
| Agent 4 | 5.5× | 2.0× | 1.0× |
| Agent 5 | 9.4× | 3.1× | 1.0× |

*注：倍数为相对于基线响应率的 fold-increase*

**核心观察**：
- **Agent0**：最强潜意识 token 使"lion"响应率提升**1,386.4 倍**
- **Agent5**（网络末端）：仍保持**9.4 倍**提升
- **统计显著性**：所有智能体的潜意识 token 效应均显著高于基线和随机 token（Bootstrap 10,000 次采样，p<0.05）

### 3.2 不同动物的传播效果

| 目标动物 | Agent 0 | Agent 1 | Agent 2 | Agent 3 | Agent 4 | Agent 5 |
|---------|---------|---------|---------|---------|---------|---------|
| Chimpanzee | 813.3× | 36.8× | 8.4× | 3.8× | 4.0× | 4.4× |
| Orangutan | 658.0× | 61.1× | 16.7× | 10.5× | 7.2× | 5.5× |
| Lion | 65.6× | 24.7× | 23.9× | 7.5× | 5.5× | 6.3× |
| Panda | 15.9× | 3.6× | 2.4× | 2.0× | 2.1× | 1.4× |

*表格显示最强潜意识 token 相对于随机 token 的 fold-increase（括号内为平均 token 效应）*

**关键发现**：
- 10 种动物中，8 种在 Agent5 仍显示显著效应
- 最大传播效应：Chimpanzee 在 Agent0 达**813.3 倍**
- 即使在与 Agent0 相隔 5 跳的 Agent5，效应仍可达**7.2 倍**

### 3.3 真实性对齐（TruthfulQA）结果

**实验设计**：
- 识别"真实性 token"（诱导最大真实性行为）和"欺骗性 token"（诱导最小真实性行为）
- 评估潜意识偏见对 TruthfulQA 准确率的影响

**核心结果**：
- 单个智能体的潜意识攻陷可**降低下游智能体的真实性**
- 欺骗性 token 传播导致网络整体准确率下降
- 效应随距离衰减，但在 6 智能体网络中仍保持统计显著性

### 3.4 拓扑结构对比

| 拓扑类型 | 传播效应 | 显著性 |
|---------|---------|--------|
| 单向链 | 强传播，单调衰减 | p<0.05 |
| 双向链 | 定性定量结果相似 | p<0.05 |

研究发现两种拓扑结构下结果**定性和定量上基本一致**，表明 Thought Virus 攻击对网络拓扑不敏感。

## 四、技术启示与发展方向

### 4.1 对 AI Agent 开发的启发

**1. 多智能体系统安全需重新评估**
- 现有防御机制（消息监控、改写）无法防御 Thought Virus
- 需在系统设计阶段考虑潜意识偏见传播风险

**2. 智能体隔离策略**
- 关键决策智能体应避免与未验证智能体直接通信
- 可考虑在智能体间引入"防火墙"层，检测异常偏见传播

**3. 潜意识 Token 检测**
- 开发针对语义无关 token 序列的检测工具
- 在系统提示词层面建立白名单机制

### 4.2 可直接应用的技术点

**防御建议**：
1. **智能体健康检查**：定期用 TruthfulQA 等基准测试评估智能体真实性
2. **消息熵监控**：检测智能体间消息中异常 token 分布
3. **拓扑优化**：在关键应用中采用更密集的连接拓扑（研究表明密集拓扑错误传播较少）
4. **提示词审计**：对系统提示词进行潜意识偏见扫描

### 4.3 研究局限性

- **模型范围**：仅在 Qwen2.5-7B 和 Llama-3.1-8B 上验证，更大模型效应未知
- **拓扑限制**：仅测试链式和双向链，未涉及网状、星型等复杂拓扑
- **攻击假设**：实验假设攻陷 Agent0 的系统提示词访问权限，实际场景中可能受限

## 五、总结

### 核心贡献

| 贡献点 | 描述 |
|-------|------|
| 新型攻击向量 | 提出 Thought Virus，利用潜意识提示传播偏见 |
| 逃避现有防御 | 无需精确措辞（逃避改写），无显式恶意内容（逃避检测） |
| 实证验证 | 在 6 智能体网络中验证偏见传播，最远 5 跳仍显著 |
| 对齐风险 | 证明单个智能体攻陷可降低整个网络真实性 |

### 实际效果汇总

| 指标 | Agent 0 | Agent 2 | Agent 5 |
|-----|---------|---------|---------|
| 最强 token 效应 | 1,386.4× | 23.9× | 9.4× |
| 平均 token 效应 | 427.0× | 5.0× | 3.1× |
| 显著性（vs 随机） | p<0.05 | p<0.05 | p<0.05 |

### 安全建议

对于部署多智能体系统的组织，建议：
1. **审计现有系统**：检查智能体间通信是否存在潜意识偏见传播风险
2. **建立监控机制**：定期评估智能体输出的一致性和真实性
3. **限制系统提示词访问**：最小化可修改智能体系统提示的权限
4. **关注后续研究**：该领域处于早期阶段，需持续跟踪最新防御方案
