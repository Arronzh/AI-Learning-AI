## 一、研究背景与动机

大语言模型的持续学习能力仍是当前 AI 领域的核心挑战。现有模型虽然在即时预测和上下文学习方面表现优异，但它们缺乏将临时的上下文知识有效迁移到长期参数中的能力。简单来说，模型学得新东西，但很快就忘了——这与人脑的运作方式形成了鲜明对比。

人类在睡眠中会经历记忆巩固（Memory Consolidation）过程：白天获取的短期、易碎记忆会在睡眠中被反复回放，最终转化为稳定的长期记忆。这一生物机制启发了研究者——**能否让 AI 模型也"睡觉"，从而自主学习并巩固记忆？**

与此同时，奖励模型（Reward Model）的评价标准长期处于碎片化状态，规则验证器、真实参考答案、过程检查清单、复杂评分标准等异构标准缺乏统一框架；大语言模型中 KV 缓存的显存瓶颈随着推理模型链式思维长度的增加日益严重；多 epoch 训练在高质量文本数据增长放缓和算力快速扩张之间的矛盾中面临边际收益递减的困境。这些问题共同构成了今日论文的核心关切。

## 二、核心方法

### 🛌 论文 1：Language Models Need Sleep（语言模型需要睡眠）

- **论文标题**：Language Models Need Sleep: Learning to Self-Modify and Consolidate Memories
- **作者**：Ali Behrouz 等
- **链接**：https://arxiv.org/abs/2606.03979

该研究提出的"Sleep"范式包含两个关键阶段：

**阶段一：记忆巩固（Memory Consolidation）**。研究者提出了一种称为"知识播种"（Knowledge Seeding）的自底向上蒸馏过程——较小模型（"小我"）中存储的临时记忆被蒸馏到更大的网络中，在保留知识的同时提供更大的容量。具体实现上，他们提出了"广义蒸馏"方法，结合了同策略蒸馏（on-policy distillation）和基于强化学习的模仿学习。

**阶段二：做梦（Dreaming）**。模型进入自我提升阶段，利用强化学习自主生成合成数据的课程式序列（curriculum），以此复述新知识并在无需人类监督的情况下精炼已有能力。这是一个递归的自改进循环。

实验覆盖了长程任务、持续学习、知识融合和少样本泛化等多个维度，结果支持了"睡眠阶段"对模型能力提升的关键作用。

### 🏆 论文 2：Skill-RM — 用 Agent Skill 统一奖励评价

- **论文标题**：Unifying Heterogeneous Evaluation Criteria via Agent Skill
- **作者**：Tao Chen, Gangwei Jiang, Pengyu Cheng 等
- **链接**：https://arxiv.org/abs/2606.03980

当前 LLM 的后训练（如强化微调 RFT 和 RL 管线）高度依赖奖励模型的反馈信号。然而，奖励评价长期依赖规则验证器、真实参考答案、过程检查清单和复杂评分标准等异构标准，缺乏统一的集成机制。

Skill-RM 的核心洞察是：**将奖励计算重新定义为执行一个可复用的"奖励评价技能"（Reward-Evaluation Skill）**。通过将奖励建模转化为结构化的 Agent 任务，Skill-RM 提供了一致的接口来编排异构资源，动态选择和聚合适配于每个输入特定需求的证据。这使得奖励模型超越了静态评价，在不同任务间保持了一致性和透明度。在奖励基准测试以及 Best-of-N 选择、强化学习等下游应用中，Skill-RM 一致优于传统方法。

### ⚡ 论文 3：VaSE — 推理模型的高效 KV 缓存淘汰

- **论文标题**：Value-Aware Stochastic KV Cache Eviction for Reasoning Models
- **作者**：Deqing Fu 等
- **链接**：https://arxiv.org/abs/2606.03928

推理模型通过长链式思维（chain-of-thought）提升准确性，但这也带来了巨大的显存和计算瓶颈。现有的 KV 缓存淘汰方法虽然减少了开销，但精度往往不如保留完整缓存的稀疏注意力选择方案。

该研究识别了两个关键因素：（1）一小部分值状态具有异常大的幅度，淘汰它们会导致灾难性失败——模型陷入重复推理循环；（2）在淘汰过程中引入随机性可通过增加缓存多样性提升精度。

基于此，VaSE（Value-aware Stochastic KV Cache Eviction）提出了无需重新训练即可应用的方案，保护大幅度值状态并促进多样化的淘汰决策。在 Qwen3 模型上，VaSE 在 4 倍 KV 缓存压缩下实现了比最先进的淘汰方法高出 4% 以上的平均准确率，同时支持 FlashAttention2，为推理模型提供了静态显存占用的可能。

### 🔄 论文 4：q0 — 超时代预训练的原语

- **论文标题**：q0: Primitives for Hyper-Epoch Pretraining
- **作者**：Bishwas Mandal 等
- **链接**：https://arxiv.org/abs/2606.03938

多 epoch 训练正在成为新常态，因为算力的增长已快于高质量文本的供给。但单个模型在几轮训练后就会饱和，远在算力预算耗尽之前。该研究提出概念性转变：**从训练单个模型转向探索模型种群并聚合它们的预测**。

q0 引入了三个核心原语：
1. **循环调度**——利用反相关的学习率和权重衰减从少数并行轨迹中收集多样化模型；
2. **链式蒸馏**——每个模型对前一模型进行蒸馏，使得模型质量在种群中累积提升；
3. **学习先验**——在保留集上拟合，为任意推理预算选择和加权成员。

在 1.8B 参数的模型上，q0 仅用约 56 epoch 就匹敌了 256 epoch 的强集成基线（约 4.6 倍数据效率提升），并在此之上继续改进。在 Slowrun 设置下累计数据效率可达约 12.9 倍。

### 🎯 论文 5：MAdam — 度量感知的多目标 Adam

- **论文标题**：MAdam: Metric-Aware Multi-Objective Adam
- **作者**：Fengbei Liu 等
- **链接**：https://arxiv.org/abs/2606.03904

多目标优化（MOO）是众多机器学习问题的基础，但现有的 MOO 求解器几乎都将其"调和后的方向"交给 Adam 优化器执行。该研究揭示了这个过程中的两个系统性偏差：

**权重偏差**：Adam 的二阶矩分母将时变的偏好向量与梯度统计纠缠在一起，把偏好退化为历史平均值，使不同的帕累托权衡坍缩为近乎均匀的混合。

**几何偏差**：Adam 的自适应度量扭曲了 MOO 求解器假设的欧几里得几何，将原本一致的目标变成了表面上的冲突。

MAdam 作为一个即插即用的包装器，通过对标量目标的偏好条件曲率对调和方向进行预处理——在这个白化后的输入上，Adam 的二阶矩坍缩为单位阵，实际更新由偏好条件度量主导。在多项任务中，MAdam 对每个求解器家族都一致优于 Adam。

### 🔬 论文 6：FFR — 将 Forward-Forward 学习扩展到回归

- **论文标题**：FFR: Forward-Forward Learning for Regression
- **作者**：Xinyang Liu 等
- **链接**：https://arxiv.org/abs/2606.03927

Forward-Forward（FF）算法以其局部、逐层优化的特性，提供了比反向传播（BP）更具生物合理性和计算效率的替代方案。但 FF 天然设计用于分类——依赖正负对比样本对，将其扩展到回归面临根本性挑战：连续目标空间缺乏自然的"反例"。

FFR 提出了三项关键创新：
1. 排序竞争性 goodness 函数——用分区神经元群之间的竞争学习替代对比对，融入距离感知的序数监督；
2. 分层阶梯架构——浅层学习粗略的序数判别，深层精化为细粒度回归，并通过多尺度特征聚合实现层间协作；
3. 层级预测与不确定性估计——多尺度预测器联合提供鲁棒预测和置信度（"免费午餐"）。

实验表明，FFR 在五个真实回归基准上恢复了 BP 平均 98.6% 的精度，同时将峰值训练内存降至 BP 的 27%（深度 8 层）和 8%（深度 32 层），每次迭代时间约为 BP 的 72%。

## 四、技术启示与发展方向

这六项工作从不同维度揭示了当前 AI 研究的前沿脉搏：

**生物启发范式正在回归。** "Sleep"范式模仿人类睡眠中的记忆巩固机制，代表了神经科学与 AI 交叉方向的又一次重要尝试。这一方向如果被验证有效，可能彻底改变模型训练后的知识管理方式——从"一次训练、静态部署"走向"学习-巩固-做梦-进化"的循环。

**奖励建模走向 Agent 化。** Skill-RM 将奖励计算重构为 Agent 技能执行，预示着奖励模型将从静态评分函数进化为动态、可组合的智能体——这是 RLHF 管线走向成熟的重要一步。

**推理效率仍是硬骨头。** VaSE 解决了 KV 缓存淘汰在推理模型中的关键痛点，保护大幅度值状态的发现对推理模型的稳定性至关重要。随着推理模型在实际应用中越来越普遍，这类方法的价值只会越来越大。

**训练范式的根本性反思。** q0 提出的"种群模型替代单一模型"思路，与当前"花更多算力训更大的模型"的主流叙事形成对比，暗示着另一种可能：多样化模型种群的协同可能比单一巨兽更具效率。

**优化器理论的精细化。** 无论是 MAdam 对 Adam 偏差的揭示，还是 Muon 动量谱滤波的理论分析，都表明优化器研究正在从"好用就行"走向"理解为什么好用"的深度阶段。

**无 BP 学习继续推进。** FFR 将 Forward-Forward 学习扩展到回归任务，以极小精度损失换取大幅内存节省，为边缘设备和生物计算打开了新可能。
