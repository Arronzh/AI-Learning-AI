# AI 论文精读｜GRPO 替代者来了？VIMPO 无评论家策略优化、DiffusionGemma 可解释性、Lie-代数注意力机制

📌 **核心摘要**：本期精选 5 篇 2026 年 6 月 19 日 arXiv 新论文，涵盖 LLM 强化学习推理优化、扩散语言模型可解释性、全新注意力机制、LoRA 合并预测、以及 KAN 与储备池计算的融合。其中 VIMPO 在数学推理基准上全面超过 GRPO，Lie-Algebra Attention 首次将 token 定义为矩阵 Lie 群元素实现等变注意力，均具有重要的技术突破意义。

---

## 一、研究背景与动机

本周五的 arXiv 新论文数量可观——仅 cs.AI 方向就新增 73 篇稿件，cs.LG 方向新增 122 篇。在 LLM 推理能力提升仍是最热门方向之一的当下，几篇论文格外引人注目：

- **强化学习与 LLM 推理**：GRPO 凭借其无需评论家的简洁训练框架推动了 DeepSeek-R1 等模型的成功，但其对每个 token 分配相同的轨迹级优势信号存在天然的信用分配局限。
- **扩散语言模型**：Google DeepMind 的 DiffusionGemma 展示了扩散模型在语言任务上的巨大潜力，但其在连续潜空间中完成大部分计算的特性引发了一个关键疑问：它的推理过程是否比自回归模型更不透明？
- **注意力机制创新**：传统的 Transformer 注意力依赖学习得到的核函数来编码 token 间关系，对于具有对称性的结构化数据（如图形、3D 场景），能否用几何先验替代学习？
- **模型合并与微调**：LoRA 已经成为高效微调的标准方案，但两个独立训练的 LoRA 适配器是否可合并，往往要等到训练完成后才能知道——这需要更早的预测信号。
- **储备池计算**：作为一种轻量级时序预测框架，储备池计算在捕捉长程依赖方面存在局限，而 KAN 网络虽表达能力强大但训练成本高，两者能否取长补短？

## 二、核心方法

### 1️⃣ VIMPO：从最优性条件中推导价值函数

**论文**：*VIMPO: Value-Implicit Policy Optimization for LLMs*
**作者**：Zhewei Kang, Aosong Feng, Sergey Levine, Dawn Song, Xuandong Zhao

VIMPO 的核心洞察来自 KL 正则化强化学习的最优性条件。在 KL 正则化的策略优化中，最优策略的解析形式满足：

$$π^*(a_t|s_t) ∝ π_{ref}(a_t|s_t) · exp(Q^*(s_t, a_t)/β)$$

VIMPO 的作者注意到，价值函数可以从策略本身与参考策略的 log-ratio 中"反向推导"出来。对于自回归生成，这形成了一个递归关系：

$$V(s_t) = \log\frac{π(a_t|s_t)}{π_{ref}(a_t|s_t)} + V(s_{t+1})$$

加上生成结束时「未来无奖励」的终止条件 $V(s_T) = 0$，就能从最终的奖励信号反向传播计算出每一步的状态价值，**完全不需要训练评论家网络**。

训练包含两个损失项：
- **价值损失**：让从 log-ratio 递归得出的价值去拟合可验证奖励（verifiable rewards）
- **Actor 损失**：用 PPO 风格的优势函数更新策略

这种设计的巧妙之处在于，它分离了「奖励嵌入」（通过价值损失）和「策略改进」（通过 Actor 损失），使得每个 token 都能获得细粒度的信用分配，同时避免了 Actor-Critic 方法中价值函数不稳定的问题。

### 2️⃣ DiffusionGemma 的可解释性探究

**论文**：*How Transparent is DiffusionGemma?*
**作者**：Joshua Engels, Callum McDougall, Bilal Chughtai, Neel Nanda 等（Google DeepMind）

这是一个来自 DeepMind 的大型可解释性研究。论文将透明性分解为两个维度：

- **变量透明性**：能否理解模型计算状态的中间快照？
- **算法透明性**：能否用这些快照重建模型得出输出的完整过程？

**发现一：变量透明性被高估了**
DiffusionGemma 的「不透明串行深度」（连续可解释状态之间的计算量）表面上是自回归 Gemma 4 的 **28.6 倍**。但研究团队发现，通过一个可解释的 token 瓶颈（token bottleneck）来映射去噪步骤之间的信息流，可以在不影响下游性能的前提下将不透明深度降低到仅 1.1 倍。这意味着扩散模型的中间表示虽然在视觉上难以解读，但信息流仍然可以通过 token 级别的瓶颈进行追踪。

**发现二：算法透明性面临根本性挑战**
在扩散模型中，所有 token 在每个去噪步骤都可以同时改变——这使得模型有能力在去噪过程中实现复杂的分布式算法。研究团队发现了若干扩散特有的现象：
- **非时间顺序推理**：模型可能在较早的去噪步骤中就"预判"了最终输出
- **token 和序列模糊化**：中间表示中 token 的身份信息被"涂抹"
- **中间上下文推理**：模型通过中间表示状态进行推理

**发现三：可监控性相当**
在评估模型输出的可监控性（monitorability）时，DiffusionGemma 与 Gemma 4 表现相当，这是一个积极信号。

### 3️⃣ Lie-Algebra Attention：Token 即群元素

**论文**：*The Token Is a Group Element: On Lie-Algebra Attention over Matrix Lie Groups*
**作者**：Przemyslaw Musialski

这篇论文提出了一个根本性的新思路：**把 token 定义为矩阵 Lie 群的元素**。每个 token 就是一个纯粹的变换矩阵，没有特征载荷，也没有外部表示 $\rho(g)$ 来承载它。

这是首个 token 为裸矩阵 Lie 群元素的注意力结构。其核心设计：

- **几何关系是规范的**：两个 token 的相对几何就是 $g_i^{-1}g_j$，天然具有等变性
- **注意力分数是封闭形式的**：$s_{ij} = -\|\log(g_i^{-1}g_j)\|^2_λ/τ$，即相对位姿的代数范数，完全不需要学习
- **G-对角作用的等变性是自动成立的**：cocycle 条件自动满足

关键突破在于：该方法适用于**任何矩阵 Lie 群**，包括包含缩放和平移的非紧致非交换仿射群——这是所有基于不可约表示（irrep）或满射指数映射方法都无法触及的。

在 SE(2)、SO(3) 和 Aff(2) 上的序列补全实验验证了：封闭形式的分数匹配了学习得到的 MLP 核的性能，在 SE(2) 上甚至更好，参数节省 **50-80 倍**；而传统的向量 token 基线在等变性上偏差了 5-12 个数量级。

### 4️⃣ MergeProbe：在训练早期预测 LoRA 可合并性

**论文**：*MergeProbe: Anticipating Adapter Mergeability Before Full Training*
**来源**：arXiv:2606.19549

论文提出了一个直击要害的问题：两个训练得很好的 LoRA 适配器，合并后的效果能否从训练初期的信号预判？

MergeProbe 定义适配器可合并性为「合并后保留单任务效用的程度」，并识别出三个早期预测信号：
1. **低秩更新方向的对齐程度**
2. **梯度的跨任务对齐**
3. **对共享表示的扰动程度**

在 MERGE-PEFT 基准（覆盖数学、代码、科学、指令跟随、安全五个领域）上，MergeProbe 在平均和最差保留率上均优于强干扰感知合并基线，同时部署开销远低于完整的任务路由方案。

### 5️⃣ KARC：KAN 与储备池计算的融合

**论文**：*Kolmogorov-Arnold Reservoir Computing*
**作者**：Juntian Huang, Jurgen Kurths, Ying Tang

KARC 用 Kolmogorov-Arnold 表示定理启发的显式基函数展开替代了传统储备池。论文严格证明了 KARC 是 KAN 网络的一种轻量级实现——保留了 KAN 的表达能力，同时支持储备池计算的高效闭式训练。

在包括偏微分方程在内的挑战性基准上，KARC 在相近的计算成本下全面优于现有储备池方法。它还可以与生成扩散模型集成用于文生图任务，架起了储备池计算与 KAN 之间的理论桥梁。

## 三、实验结果

### VIMPO 实验结果

| 基准 | GRPO | VIMPO | 提升 |
|------|------|-------|------|
| MATH-500 | 基准线 | 超越 | 显著 |
| AIME 2024 | 基准线 | 超越 | 显著 |
| AIME 2025 | 基准线 | 显著超越 | 竞赛级最大增益 |
| OlympiadBench | 基准线 | 超越 | 显著 |

在噪声奖励条件下，VIMPO 保持了相对于 GRPO 的一致优势，表明策略隐含价值优化在保留无评论家训练简洁性的同时，提供了更细粒度的信用分配。

### DiffusionGemma 关键数据

- **表面不透明串行深度**：28.6×（vs 自回归 Gemma 4）
- **使用 token bottleneck 后**：1.1×（几乎无差异）
- **可监控性**：与 Gemma 4 相当

### Lie-Algebra Attention 实验结果

- SE(2) 序列补全：封闭形式分数优于学习核
- SO(3) 序列补全：匹配学习核性能
- Aff(2) 序列补全：成功处理含缩放和剪切的仿射群
- 参数效率：**50-80 倍**参数减少
- 等变性：向量 token 基线的等变性误差高达 5-12 个数量级

### MergeProbe 实验结果

在 MERGE-PEFT 五领域基准上，MergeProbe 在平均保留率和最差保留率上均优于所有基线，同时避免了全任务路由的高部署成本。

### KARC 实验结果

在偏微分方程预测等挑战性任务上，KARC 以相近计算成本超越了传统储备池计算和下一代储备池计算方法。

## 四、技术启示与发展方向

### 1. GRPO 之后的路径
VIMPO 的出现标志着一个重要信号：**GRPO 虽好，但并非终点**。通过从最优性条件中解析推导价值函数，VIMPO 在不增加训练复杂度的前提下实现了更好的信用分配。这意味着在 LLM 推理增强领域，我们可能在不久的将来看到更多利用 RL 理论中经典结果的新方法。

### 2. 扩散 LLM 的透明性
DeepMind 的这项工作表明扩散语言模型的可解释性虽然面临独特挑战，但也并非无法解决。**token bottleneck 的发现**尤为实用——它提供了一个通用的调试工具。非时间顺序推理等扩散特有现象的发现，也提示我们需要发展新的可解释性方法，而不是简单移植自回归模型的分析工具。

### 3. 几何先验的回归
Lie-Algebra Attention 体现了用**结构先验替代数据驱动学习**的趋势。对于具有明确几何对称性的问题（机器人、图形学、分子建模），这种先验带来的不仅仅是参数效率的提升，更重要的是等变性保证。这项工作也可能启发更多将 token 嵌入到非欧空间的研究。

### 4. 从后验到先验的模型合并
MergeProbe 让我们可以在训练早期就预判 LoRA 适配器的合并效果——**将工程决策从实验后分析变为实验前预测**。这对于大规模适配器管理和多任务部署具有实际意义。

### 5. KAN 的工程化落地
KARC 证明了 KAN 理论可以落地为高效的实际算法，而不是停留在理论优雅层面。将 KAN 与储备池计算结合，既能利用 KAN 的表达能力，又能使用储备池计算的闭式解训练，是一个值得关注的方向。

## 五、总结

本期 arXiv 论文呈现出几个清晰的技术趋势：

1. **LLM 推理增强**继续是焦点——VIMPO 展示了如何在无评论家框架下实现细粒度信用分配
2. **可解释性研究进入新的阵地**——从自回归模型扩展到扩散模型，并发现了独特的扩散特有现象
3. **几何深度学习**有了基础性的注意力机制创新——Lie-Algebra Attention 将 token 嵌入到矩阵 Lie 群中
4. **实用主义创新**——MergeProbe 和 KARC 都在现有框架上找到了巧妙的改进点

这些工作共同指向一个方向：**AI 研究正在从"能不能做"转向"能不能理解、能不能控制、能不能高效"**。
